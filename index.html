Why you should choose KodeMan?

KodePas - the lightest SDK for coders

KodePas can be the lightest SDK with only 100 MB. KodePas provides you a CLI interface, some great tools that Kode Organization build from the GNU softwares and some other is built from scratch will work perfectly out of the box such as: KGet - a tools from Wget, KodePas Rendering - store files into a package with highly encrypted.



KodeMan - the lightweight and includes minimal features for you

KodeMan gives you a graphical UI, with fast, friendly and easy to use. You only need to have a machine that can use KodePas. Also, KodeMan gives you information about KodePas sites, Wiki and little information about Kode Organization. Although you will need to install some external softwares (some platforms), but it is still lightweight
